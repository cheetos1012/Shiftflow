# ⚡ ShiftFlow

https://shiftflowww.netlify.app/
Современный планировщик задач с тёмной темой и красивым дизайном.

## ✨ Возможности

- Добавление задач с текстом и датой/временем
- Отметка выполненных задач
- Удаление задач
- Фильтры: Все / Активные / Выполненные
- Статистика и прогресс-бар на сегодня
- Подсветка просроченных задач
- Автосохранение в браузере (localStorage)
- Адаптивный дизайн для телефона и ПК
- Анимации и плавные переходы

## 🚀 Как запустить

Просто открой файл `index.html` в любом браузере.

Или склонируй репозиторий:
```bash
git clone https://github.com/cheetos1012/shiftflow.git
cd shiftflow
open index.html
```
